# Relay-selection
Source code of learning-based relay selection algorithms in the research "Optimizing Relay Selection and Wireless Resource Allocation in Proximity Radio Access Networks"
# Environment
The learning based relay selection program was implemented on TensorFlow 2.1.0 with python 3.7.
# How to train
```
python DecisionTree.py
Python RandomForest.py
python XGBoost.py
```
